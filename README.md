# Семейное древо Андросовых

Интерактивный семейный сайт, задеплоенный на GitHub Pages.

## 🌐 Открыть сайт

После деплоя сайт будет доступен по адресу:  
`https://[твой-username].github.io/androsov-family/`

## 📁 Файлы

- `index.html` — весь сайт в одном файле

## 🚀 Как задеплоить на GitHub Pages

1. Создай новый репозиторий на [github.com/new](https://github.com/new)
   - Назови его, например: `androsov-family`
   - Сделай его **публичным** (Public)

2. Загрузи файлы:
```bash
git init
git add .
git commit -m "Семейное древо Андросовых"
git branch -M main
git remote add origin https://github.com/[USERNAME]/androsov-family.git
git push -u origin main
```

3. Включи GitHub Pages:
   - Зайди в репозиторий → **Settings** → **Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **(root)**
   - Нажми **Save**

4. Через 1–2 минуты сайт будет доступен! 🎉
